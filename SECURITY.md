# Security

Report vulnerabilities privately. Do not open a public issue for exploits or sensitive crash details.

**Email:** info@quietsync.net  
**Project:** https://github.com/macdirtycow/openoriel

Include OS version, Oriel build, steps to reproduce, and impact. We will acknowledge mail when we can and coordinate a fix before any disclosure.

Oriel runs untrusted web content inside WebKit. Reports that only demonstrate normal web-page behavior (phishing pages, malicious JS within the page sandbox, etc.) are usually out of scope unless they cross into native code, the sandbox, or Oriel’s bridges.
