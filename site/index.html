Inside the same app

Oriel Pulse

A gaming-inspired edition you can switch on anytime — neon chrome, Pulse Corner, performance limits, Data Saver, ambience, and Play / Stream / Focus workspace presets. Same privacy model as Classic Oriel. No Opera-style account. No fake CPU meters.

  • Pulse Corner quick launch (Twitch, YouTube, Discord, Steam) + gaming news
  • Honest engine limits + Data/Network Saver + Lucid Mode + Low Power Mode
  • Local soundscape and start-page wallpapers — nothing streamed or sold
  • Optional Pulse home-screen / Dock icon
  • Same Mac page-engine picker as Classic (WebKit / Chromium Compatible)
  • Mac: auto Chromium Compatible for stubborn sites, per-site overrides, Client Hints

Turn it on in Settings → Appearance after install. Switch back to Classic whenever you want. Page engine works in both editions.

Best of both engines

On Mac, the default is Smart (best per tab): each tab picks WebKit or Chromium Compatible from the page (Meet/Teams-style apps get Chrome identity; Apple/captcha-sensitive sites stay on WebKit). You can also lock every tab to WebKit or Compatible. Real Chromium Native arrives when a CEF build ships. iPhone and iPad stay on WebKit — Apple’s rule, stated honestly.

Native on purpose

Oriel is Apple-first: process-shared WebKit, system materials, and chrome that stays out of the way so the page can breathe. Chromium Compatible is an opt-in Mac identity layer — not a fake “we shipped Blink on iPhone” claim.

Oriel Shields

Bundled content-blocker lists — EasyList, AdGuard-derived filters, DuckDuckGo tracker hosts — plus cosmetics and YouTube help. Honest about WebKit limits: some first-party stacks still slip through.
